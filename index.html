Quick Theory Reference

Homogeneous Equation: M(x,y)dx + N(x,y)dy = 0 is homogeneous if M(tx,ty) = tⁿM(x,y) and N(tx,ty) = tⁿN(x,y) for some n.

Exactness Condition: ∂M/∂y = ∂N/∂x for exact differential equations.
